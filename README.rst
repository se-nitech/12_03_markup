コード
=========

コードを入力する場合は以下のようなコードブロックを使います．

.. code-block:: python

    def myfunc(a, b, c):
        a = b + c
        print(a)
    


数式
=========

レンダラによっては :math:`y = f(x)` のようなLaTeX数式も対応しています．インラインだけでなく，行立ての数式

.. math::

    y = \int_a^b f(x, t) dt
    
も対応しています．
